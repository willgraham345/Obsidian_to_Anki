If else statements can be condensed as shown

```cpp 
variable = (condition) ? condition_is_true : condition_is_false;
```

Curly brackets can be omitted if there is only one statement following the condition 

```cpp
if (number == 7)
  std::cout << "Lucky!\n";

```
