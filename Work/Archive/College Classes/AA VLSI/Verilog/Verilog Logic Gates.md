# Basic ones
| Gate | Symbol                         | Description              |
| ---- | ------------------------------ | ------------------------ |
| AND  | `.` or `*` (sometimes omitted) | Output 1 if inputs are 1 |
| OR   | `+`                            | Output if any input is 1 |
| NOT  | `'` or $\bar{A}$               | Output if nothing is 1   |
| XOR  | `^` or $^\wedge$               | Output of 1 if inputs are different                         |
