# `saveImageToFile()`
Overview:
- Saves image to the file, if it isn't already saved it will call [[VD.VivaPro.ImageDataContainer.saveAsImageToFile()]]
- 
Operations:
1. Checks to see if it can open the file
2. Saves header
3. Saves standard image
